OLÁ, TUDO BEM?

O Hangman Game foi desenvolvido em janeiro de 2025 por mim, Hugo Araujo.

Este jogo é o resultado dos meus estudos em programação web. Foi feito com carinho para testar minhas habilidades (e o seu vocabulário!).

Atualmente, o sistema consome dados dinamicamente, com todas as palavras armazenadas em um banco MongoDB.
